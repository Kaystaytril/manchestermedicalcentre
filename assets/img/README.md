# Images

Drop the clinic's real photos in this folder, then swap them into the pages
where you see a `<!-- REPLACE PHOTO: ... -->` comment (see `../../CONTENT.md`,
section 4).

Suggested files (any of these — use what you have):

| Filename | Used on | Suggested size |
|---|---|---|
| `reception.jpg` | Home hero, About | ~1200 × 1000 px, landscape |
| `interior.jpg` | Home "why us" | ~1200 × 900 px, landscape |
| `optometry.jpg` | Eye Care page | ~1200 × 900 px |
| `dentistry.jpg` | Dental page | ~1200 × 900 px |
| `physiotherapy.jpg` | Physiotherapy page | ~1200 × 900 px |
| `optometrist.jpg` | Team + Eye Care | square, ~800 × 800 px |
| `dentist.jpg` | Team + Dental | square, ~800 × 800 px |
| `physiotherapist.jpg` | Team + Physiotherapy | square, ~800 × 800 px |

Tips:
- **JPEG** for photos, **PNG/SVG** for logos or graphics.
- Compress before uploading (e.g. squoosh.app) so pages stay fast — aim for
  under ~300 KB per photo.
- Always add a descriptive `alt` attribute when you insert an image, for
  accessibility and SEO.
- You have permission to use the clinic's own photography; please make sure you
  hold the rights to any other images you add.
